Doc Generator

Generates HTML from markdown files in local files or GitHub repositories.

Functions

less

Converts a LESS file to CSS.

Parameters

  • An object with properties:
    • inputFolder: folder containing the input file
    • inputFilename: name of the input file
    • outputFolder: folder where to place the output file
    • outputFilename: name of out put file
  • Callback function

Example of use

const generator = require('doc-generator');
generator.less({
  inputFolder : 'less',
  inputFilename : 'index.less',
  outputFolder : 'shared/css',
  outputFilename : 'style.css',
})

lessList

Converts several LESS files to CSS files.

Parameters

  • A list of objects with properties:
    • inputFolder: folder containing the input file
    • inputFilename: name of the input file
    • outputFolder: folder where to place the output file
    • outputFilename: name of out put file
  • Callback function

Example of use

const generator = require('doc-generator');
generator.lessList([
  {
    inputFolder : 'less',
    inputFilename : 'index.less',
    outputFolder : 'shared/css',
    outputFilename : 'style.css',
  },
  {
    inputFolder : 'less',
    inputFilename : 'print.less',
    outputFolder : 'shared/css',
    outputFilename : 'print.css',
  },
])

convert

Converts a Markdown file to HTML.
